Module Import
Constant

This module provides constants of fitness and health data types.

Start version: 5.0.0(12)
Module Import
Supported devicesPhoneTabletWearable

    import { healthStore } from '@kit.HealthServiceKit';

NOTE

This module is a sub-module of healthStore and needs to be used in healthStore.healthDataTypes mode.
