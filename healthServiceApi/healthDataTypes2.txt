healthFields (Fitness and Health Data Fields)
Last updated: 2025-11-21 03:47
Content overview
Expand

This module provides a list of fitness and health data fields.

Start version: 5.0.0(12)
Module Import
Supported devicesPhoneTabletWearable

    import { healthStore } from '@kit.HealthServiceKit';

NOTE

This module is a sub-module of healthStore and needs to be used in healthStore.healthFields mode.

number
	

No
	

Yes
	

Whether the user stands. If there is an activity record in an hour, the first minute of the hour is marked as 1. The value is left empty by default.

The options are:

0: no

1: yes
